# v0.1.0 – Experimental public release

First public experimental release of this repository.

## Current first-party path

- ChatGPT Projects
- Safari on macOS
- Codex CLI

This repository is intentionally narrow and environment-dependent.
It is not a generic browser automation framework.

## Included in this release

- `start_bridge.py` as the normal entry point
- user-authored first request flow
- same-chat default continuation
- late-completion-aware handoff / new-chat preprocessing
- soft-success wait handling for project-page handoff
- OSS-facing README / CONTRIBUTING / SECURITY / publishing checklist
- minimal Python CI
- MIT license

## Warnings

- Experimental
- Strongly dependent on Safari, ChatGPT UI, macOS Automation, and local Codex CLI behavior
- Unsupported combinations may or may not work
- Use at your own risk
- No warranty

## Notes

Other browsers, other AI tools, or other integration surfaces may be added later, but they are not currently guaranteed or maintained.
