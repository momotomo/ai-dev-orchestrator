# Copilot Instructions

This repository is bootstrapped for `ai-dev-orchestrator`.

## Required Reading

- Read `AGENTS.md` before proposing multi-file or behavior-changing edits.
- Treat `AGENTS.md` as the source of truth for repo-specific working rules.

## Delivery Rules

- Keep the requested change bounded to the named issue, prompt, or review
  slice.
- Prefer concrete file edits over broad rewrites or speculative cleanup.
- Preserve existing conventions unless the task explicitly asks for a change.

## Verification Rules

- Name the automated check you ran for the touched area.
- If you could not run a check, say that clearly instead of implying it passed.
- Stop after writing the requested files and report the paths you changed.
